JavaScript 函数作用域

  • Javascript

JavaScript 的作用域通过函数来定义的，在一个函数中定义的变量只对这个函数内部可见，称为函数作用域。在函数中引用一个变量时，JavaScript 会先搜索当前函数作用域，如果没找到才搜索其上层作用域。

这是 JavaScript 的一个特性，按照作用域搜索顺序。当 console.log 访问到 scope 时，Javascript 先会搜索函数内 fun 内的作用域，此时作用域里面能搜到 scope，所以上层作用域中定义的 scope 被屏蔽。

但当执行到 console.log 时，此时 scope 还未定义，所以得到 undefined 值。


总结：引用函数内变量，现在函数作用域内寻找有无该变量， 找到后看执行到该变量时，该变量有无定义。
