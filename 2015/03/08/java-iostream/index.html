Java 字节流

  • Java

字节流

字节流是指基本单位是一个字节的信息系列。信息在计算机中都是以8位二进制————即一个字节来处理，所以可以将各种信息用字节流统一处理。

按照基本数据类型读写：DataInputStream 和 DataOutputStream。这两个类属于过滤流。将基本字节输入输出流自动转换成基本数据类型进行读写。



输出流：

数据 ———> DataOutputStream(往输入流中写入Java基本数据类型) ———> BufferedOutputStream(提供数据写入到缓冲区功能) ———> FileOutputStream(将数据写入到文件上) ———> 文件



输入流：

文件 ———> FileInputStream(从文件中获取输入字节) ———> BufferedInputStream(增加缓冲功能) ———> DataInputStream(增加读取Java基本数据类型功能) ———> 数据


ByteArrayOutputStream与ByteArrayInputStream

ByteArrayOutputStream: 在创建实例时，程序内部创建一个byte[]数组的缓冲区，然后利用ByteArrayOutputStream的实例向数组中写入或读出byte型数据。

网络传输中需要传输很多变量，可以利用ByteArrayOutput先收集所有的变量，在一次性把数据发出去。

ByteArrayInputStream: 将字节数组转化为输入流。



结合DataInputStream与DataOutputStream过滤流对基本数据类型进行读写。


网络连接

while循环中，输入流in的read(byte[] b)方法，将读取的字节存放在buffer数组中，并返回读取的字节数，若读取完则返回-1。

过滤流ByteArrayOutputStream的write(byte[] b, int off, int len)方法, 读取字节数组b中从off开始，长度为len的字节到输出流out中。

out的toByteArray(), 将返回一个保存输出流中全部数据的字节数组。
