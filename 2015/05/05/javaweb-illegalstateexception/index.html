Java Web Cannot forward after response has been committed

  • Java

用Java Web 写网页时遇到后台服务器报错。错误提醒为 Cannot forward after response has been committed。

本来是想向对应Servlet提交请求后将结果返回到另一个jsp页面，调用了forward方法，程序如下：

但结果出现如题所述的错误，后来通过查找stackoverflow发现了问题所在。是在ImageServic的静态方法processRequess中，调用了 PrintWriter out = response.getWriter()方法，用于在页面输出相关结果用于调试。

当调用out之后，response已经响应了HTTP并进行了发送，将消息写入客户端了，故在而后的重定向中调用forward再次响应就会出现错误，因为response已经被执行了。

所以解决的办法：

  1. 在redirect 或 forward 之前不要出现流的输出。

  2. 并在redirect 或 forward 后加上 return;
