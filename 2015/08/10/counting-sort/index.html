Counting sort 计数排序

  • Algorithm

例子说明

假设排序数组为 A = {3, 1, 2, 1, 0, 3, 1}。

数组A的最大值为3，最小值为0。所以创建一个长度为 (3 - 0) + 1 的数组C，用于记录每个数字出现的次数。

扫描一趟数组A，得到数组C = {1, 3, 1, 2}

扫描一趟数组C，对C进行累加，得到数组C = {1, 4, 5, 7}

最后将数组C的计数依次输出到一个数组B中，如 B[0]=0; B[1]、B[2]、B[3]=1; B[4]=2; B[5]、B[6]=3。

数组C的下标是数组A中的值，而C[i]是该值的累加次数。
