IOS Memory Warning

  • iOS

IOS官方问当中响应内存警告的方法主要有3个

  • 实现applicationDidReceiveMemoryWarning:委托方法 —————— app委托对象接收内存警告消息

  • 在自己的UIViewController的子类中重写didReceiveMemoryWarning方法 —————— 视图控制器中接收内存警告消息

  • 通过注册UIApplicationDidReceiveMemoryWarningNotification接收消息 —————— 使用通知接收内存警告消息
