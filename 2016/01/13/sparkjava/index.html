Spark Java 轻量级Web框架

  • Java

Spark Java 是Java的一个轻量级Web框架，使得Java Web的编写变得极为简洁。由于直接封装了Servlet，使得不用像以往那样繁琐的操作Servlet实现相关功能。

最新版Spark Java支持Java 8，按照官网的介绍，Spark是一个受Ruby Web框架Sinatra启发，用Java 8以最小花费来创建Web应用的微型框架。

“Spark - A tiny Sinatra inspired framework for creating web applications in Java 8 with minimal effort”

Spark官网: http://sparkjava.com/
